我的收藏

你在各个视频里标记的句子，集中在这里复习、导出到 Anki。数据存在本机；开启云同步后可跨设备。

⚙ 云同步（可选 · 跨设备）
用一个 GitHub 个人访问令牌（classic，仅勾选 gist）把收藏同步到一个私密 gist；手机标记、电脑复习。

令牌只存在这台设备的浏览器里，永不进入公开仓库。classic gist 权限覆盖你账号下的所有 gist， 建议用一个专用/小号 GitHub 账号，并给令牌设置较短有效期（7–30 天）。“secret gist”只是不公开列出、并非加密—— 任何拿到链接的人都能看。

还没有收藏。打开任意讲解页，点句子的书签图标即可收藏。

收藏保存在本机浏览器（localStorage）。iOS Safari 会在约 7 天无访问后清理本机存储， 所以请开启云同步或定期导出备份。 · ← 返回目录
